## Educational DP Contest / DP まとめコンテスト
### A問題
https://atcoder.jp/contests/dp/tasks/dp_a
```python
N = int(input())
h = list(map(int, input().split()))

dp = [0] * N
for i in range(N):
  if i == 0:
    dp[0] = 0
  elif i == 1:
    dp[1] = abs(h[0] - h[1])
  else:
    dp[i] = min(dp[i-1] + abs(h[i-1] - h[i]), dp[i-2] + abs(h[i-2] - h[i]))

print(dp[N-1])
```
