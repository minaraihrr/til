## CommonJS モジュール→ ESM import方法（標準モジュール）
CommonJS  
```JavaScript
const xxx = require('xxx');
const key = xxx.key;
```
ES Module  
```JavaScript
import xxx from 'xxx';      // default import
const { key } = xxx;        // 分割代入でプロパティを取り出す
```
#### 別名に変更もOK  
CommonJS  
```JavaScript
const x = require('xxx');
const k = x.key;
```
ES Module  
```JavaScript
import x from 'xxx';      // default import
const { key : k } = x;    // 分割代入でプロパティを取り出す
```
